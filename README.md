# Mastermind

Java implementation of the classic **Mastermind** game, developed for the *Laboratórios de Programação* course at Faculdade de Ciências da Universidade de Lisboa.

## About the project

This project recreates a version of the Mastermind game using **object-oriented programming in Java**. The game is based on a secret code made of coloured pins, and the player must guess the correct combination within a limited number of attempts.

For each guess, the system provides feedback about:

- correct colours in the correct position
- correct colours in the wrong position

The project focuses on class interaction, arrays, enums, and game logic.

## Features

- Secret code generation
- Guess validation and evaluation
- Feedback after each play
- Score tracking
- Round management
- Support for repeated guesses handling
- Text-based game representation

## Technologies

- **Java**
- Object-Oriented Programming
- Arrays
- Enums
- JUnit tests

## Project structure

- `Code` – represents a code sequence
- `Mastermind` – manages the game state, rounds, score, and plays
- `Main` – used to run and test the game
- `Colour` – enum representing the available colours

## Learning goals

This project was developed to practise:

- interaction between classes and enums
- array manipulation
- algorithmic thinking
- game state management in Java

## How to run

1. Clone the repository
2. Open the project in your Java IDE
3. Compile the source files
4. Run the `Main` class
